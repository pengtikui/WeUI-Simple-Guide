## WeUI 简明入门指南

WeUI 是由微信官方设计团队为微信内网页和微信小程序开发的基础样式库，目的是为了微信内网页和微信小程序的用户视觉和交互体验与微信保持一致。

之前写过一篇 WeUI 入门指南，但是仅适用于 0.4.x 版本。前段时间 WeUI 发布了 1.0 版本，代码改动较大，所以重写入门指南。

> 此入门指南非微信官方文档！

喜欢或对您有帮助请 star，帮助完善文档请 pull request。:blush:
